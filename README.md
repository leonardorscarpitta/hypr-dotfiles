# Hyprland dotfiles
Esse repositório contém as dotfiles para serem utilizadass a princípio em um ambiente com **Arch Linux + Hyprland**, em outras distribuições, podem ser necessárias algumas modificações.

## Manual de instalação
1. Navegue até o seu diretório de configuração, por padrão é o `cd $USER/.config`.
2. Clone o repositório utilizando o `git clone https://github.com/leonardorscarpitta/`
3. Remova o `.git`, caso queira salvar em um repositório próprio utilizando `rm -rf .git`
4. **Bônus**: A fonte utilizada é a [SpaceMono Nerd Font](https://fonts.google.com/specimen/Space+Mono)

> [!NOTE]
> Esse repositório não instala nada em seu computador, apenas copia configurações de alguns pacotes.
